# 📄 HWP 1번 미주 기준 자동 분할 인쇄 도구

한글(HWP) 문서 내의 1번 미주(Endnote)를 자동으로 탐색하여,
이를 기준으로 본문(A3)과 답지(A4)를 서로 다른 프린터로 자동 분할 출력하는 자동화 도구입니다.

---

# 핵심 기능

## • 스마트 분할

문서 내 **첫 번째 미주 위치**를 추적하여 인쇄 범위를 자동으로 계산합니다.

## • 듀얼 프린팅

본문과 답안지를 **각각 다른 규격의 프린터**로 한 번에 전송합니다.

## • 멀티 파일 처리

폴더 내의 **모든 HWP / HWPX 파일**을 순차적으로 자동 처리합니다.

## • 안정성

* `threading`을 이용한 **UI 응답성 유지**
* **UAC 관리자 권한 대응**
* 장시간 인쇄 작업에서도 안정적으로 동작

# • 사전 프린터 설정 (필독)

프로그램의 정상적인 작동을 위해 **아래 프린터 설정을 먼저 완료해야 합니다.**

## 1️⃣ 프린터 수동 추가

1. Windows **설정 → 블루투스 및 장치 → 프린터 및 스캐너** 이동
2. **장치 추가** 버튼 클릭 후 잠시 대기
3. **원하는 프린터가 목록에 없습니다 → 수동으로 추가** 선택

## 2️⃣ IP 주소를 이용한 연결

1. **TCP/IP 주소 또는 호스트 이름으로 프린터 추가** 선택
2. 장치 유형 → **TCP/IP 장치** 선택
3. **호스트 이름 또는 IP 주소**에 프린터 고정 IP 입력

예시

```
192.168.0.100
```

4. **포트에 쿼리하여 사용할 프린터 드라이버를 자동으로 선택** 체크
5. **다음** 클릭

## 3️⃣ 프린터 이름 식별 (권장)

설치 마지막 단계에서 **프린터 이름을 구분하기 쉽게 변경**하면 편리합니다.

예시

```
본문용_A3_프린터
답지용_A4_프린터
```

프로그램 실행 시 **콤보박스에서 해당 프린터 이름을 선택**하여 사용합니다.
