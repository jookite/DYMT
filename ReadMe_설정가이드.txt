===========================================================
[필독] HWP 1번 미주 기준 자동 분할 인쇄 도구 설정 가이드
===========================================================

본 프로그램은 '1번 미주'를 기준으로 문서를 본문과 답지로 자동 분할하여 
각각 지정된 프린터로 출력하는 도구입니다. 

정상적인 작동을 위해 아래의 프린터 설정을 반드시 먼저 완료해 주세요.

1. 프린터 수동 추가 방법
   ① 윈도우 [설정] -> [블루투스 및 장치] -> [프린터 및 스캐너] 이동
   ② [장치 추가] 버튼 클릭 후 기다림
   ③ [원하는 프린터가 목록에 없습니다] 옆의 [수동으로 추가] 클릭

2. IP 주소를 이용한 연결
   ① [TCP/IP 주소 또는 호스트 이름으로 프린터 추가] 선택 후 [다음]
   ② 장치 유형: [TCP/IP 장치] 선택
   ③ 호스트 이름 또는 IP 주소: 프린터의 고정 IP 입력 (2관IP: 192.168.?.??)
   ④ [포트에 쿼리하여 사용할 프린터 드라이버를 자동으로 선택] 체크 확인 후 [다음]

3. 프린터 이름 식별 (권장)
   ① 설치 마지막 단계에서 프린터 이름을 프로그램에서 찾기 쉽게 변경합니다.
      (예: 본문용_A3_프린터 / 답지용_A4_프린터)
   ② 프로그램 실행 시 '본문(A3)'과 '답지(A4)' 콤보박스에서 해당 이름들을 선택합니다.

※ 주의사항
- PC에 한컴오피스(한글)가 반드시 설치되어 있어야 합니다.
- 인쇄 중에는 한글 프로그램이나 인쇄 창을 강제로 닫지 마세요.
